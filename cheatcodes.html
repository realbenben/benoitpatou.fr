Here's my cheat list

Cheat list ? why ?

J'ai crée cette petite liste en support car en tant que débutant, c'est important de connaitre les shortcuts essentiels et de pouvoir y avoir accès sur tout support et à tout moment.

Sublim Text

  • ctrl + z = annuler les dernières modification (appuyer plusieurs fois).
  • ctrl + shift + d = dupliquer une ligne.
  • ctrl +d + selectionner un mot = selectionne tout le mots identiques du code.
  • ctrl + / = mettre une ligne en commentaire et vice versa.
  • ctrl + shift + flèche haut et bas = déplacer une ligne.
  • ctrl + p = acceder à un fichier ou therme avec le nom.
  • ctrl + g + nombre = acceder au numéro de ligne correspondant au nombre.
  • ctrl + clic de souris = selection multiple avec la souris
  • ctrl + shift + p = acceder au menu des install.

terminal linux

  • cd = sortir du fichier ouvert
  • cd + nom du fichier = entrer dans le dossier nommé
  • ls = voir les fichiers que contient un dossier
  • ll = idem ls mais avec detail comme tille du fichier type et création
  • ctrl + c = revenir a la console
  • rm -rf + "nom du fichier" = effacer un fichier

Github via terminal

  • git status = voir sur un fichier les derniers élément non enregistré sur Github
  • git reset --"nom du fichier" = effacer un fichier de Github
  • git pull = récuperer la sauvegarde d'un fichier commun travaillé par quelqu'un d'autre
  • git remote = sert a voir l'etat d'un fichier commu travaillé par quelqu'un d'autre
  • git init = créer un repertoire sur Github
  • git clone = sert à dupliquer un fichier deja existant
  • git add -A = ajoute ou met à jour tout les fichier d'un répertoire à Github
  • git add -"nom du fichier" = ajoute ou met à jour un fichier d'un répertoire à Github
  • git commit -m "nom du commit" = sert à donner un titre à la mise a jour effectuée
  • git push = upload le fichier sur Github

Web browser

  • F12 = ouvrir la console
  • ctrl + r = recharger la page(pratique lorsque l'on code une page)
  • ctrl + shift + r = vide le cache et force le rechargement de la page
  • ctrl + tab = changer d'onglet sur le navigateur
  • ctrl + un chiffre = ouvre l'onglet numéro ....
